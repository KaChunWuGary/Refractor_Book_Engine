# Book Engine Refactor 
          
[![License: MPL 2.0](https://img.shields.io/badge/License-MPL_2.0-brightgreen.svg)](https://opensource.org/licenses/MPL-2.0)
          
## Description
This application is refactored from a RESTful AI to GraphQL API with Apollo. This is built using a MERN stack, and allows users to save book searches to the server.
